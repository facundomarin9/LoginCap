# Sistema Login

Pequeño sistema para mostrar usuarios, registrarse y loguearse. Uso de JWT para el inicio de sesion.

Tecnologías usadas: 

- Java 11
- JavaScript
- HTML (Basado en una plantilla)
- Spring
- Hibernate
- Maven
- JWT
- Argon2

IDE: Apache NetBeans
